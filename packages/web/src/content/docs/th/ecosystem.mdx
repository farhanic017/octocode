ชุดโครงการชุมชนที่สร้างขึ้นบน OctoCode

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ OctoCode ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-octocode](https://github.com/awesome-octocode/awesome-octocode) และ [octocode.cafe](https://octocode.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
